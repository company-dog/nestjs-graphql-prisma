## Prisma

### Installation

```
yarn add -D @prisma/cli
yarn add @prisima/client
```

### Init Prisma

```bash
yarn prisma init
```

### Setup Table

```bash
sqlite3 prisma/dev.db < sqlite.sql
```

### Introspect & Generagte Prisma

```bash
yarn prisma introspect
```

```bash
yarn prisma migrate
```
